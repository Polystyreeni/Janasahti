# Janasahti
## FI
Oma versio mobiilipelistä Wordz-Sanajahti. 

### Sisältö
- Neljä eri pelimuotoa
  - **Normaali**: 4x4-ruudukko, josta on etsittävä 3-10 kirjaimen pituisia sanoja. Pidemmät sanat tuottavat enemmän pisteitä. 
  - **Rationaalinen**: Kaikki sanat ovat yhtä arvokkaita. Eniten sanoja löytänyt voittaa.
  - **Aikajahti**: Peli alkaa lyhennetyllä ajalla. Löydetyt sanat tuovat lisää aikaa. Nopea sanojen löytäminen palkitsee aikabonuksilla.
  - **Laaja**: Normaali pelimuoto 5x5 ruudukolla. Sanojen pituus on enintään 16 kirjainta.
- Parhaat pelaajat -tilastot
  - Kilpaile muiden pelaajien kanssa parhaista pisteistä
  - Tilastot on mahdollista kytkeä pois päältä
- Offline tila
  - Peliä on mahdollista pelata myös ilman verkkoyhteyttä
  - Julkiset pistetilastot eivät ole käytössä offline-tilassa
 
### Käyttövaatimukset
- Android 6.0 Marshmallow (API-level 23) tai uudempi (Testattu 6.0 eteenpäin versioon 16 asti)
- Internet-yhteys (vähintään ensimmäisellä käynnistyskerralla käyttäprofiilin luontia varten)

### Kehittäjäohjeet
- Kehittäjäohjeet on kuvattu [täällä](docs/DEVELOPERS.md).

## EN
> [!NOTE]
> All content of the application is in Finnish!
My own version of the mobile game Wordz-Sanajahti.

### Features
- Four different game modes
  - **Normaali (Normal)**: 4x4 letter grid, where the player must find 3-10 letter words. Longer words yield more points.
  - **Rationaalinen (Rational)**: All words yield the same amount of points. The player with most words wins.
  - **Aikajahti (Time chase)**: The game starts with reduced time. Found words reward extra time. Time bonuses are awarded for fast actions.
  - **Laaja (Extended)**: Normal gamemode with 5x5 letter grid. Words can be up to 16 letters.
- Best player leaderboards
  - Compete with other players for the best score
  - Online leaderboards can be turned off in the settings
- Offline mode
  - Game can be played without network connection
  - Public leaderboards are not available in offline mode
 
### Usage requirements
- Android 6.0 Marshmallow (API level 23) or newer (Tested starting from 6.0 up to Android 16)
- Internet connection (for first start to register a user account)
